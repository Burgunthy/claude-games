# Claude Poker - Claude Games Plugin

Play Texas Hold'em Poker directly in Claude Code with AI opponents.

## Usage

```
/claude-games:claude-poker
```

Start a new poker game with the default settings (4 players, 1000 chips each).

## Keyboard Controls

| Key | Action |
|-----|--------|
| `F` | Fold |
| `C` | Check / Call |
| `R` | Raise |
| `A` | All-In |
| `Enter` | Start new hand / Next round |
| `Q` / `Esc` | Quit |

## Game Phases

1. **Pre-Flop** - Each player receives 2 hole cards
2. **Flop** - 3 community cards revealed
3. **Turn** - 4th community card revealed
4. **River** - 5th and final community card
5. **Showdown** - Best hand wins the pot!

## Tips

- Start by pressing `Enter` to begin
- Use `C` to check when no bet is required
- Use `C` to call when matching a bet
- Use `R` to raise the stakes
- Watch your chip stack and the pot size
- Fold weak hands early to save chips

## First Time Playing?

1. Run `/poker`
2. Press `Enter` to start the game
3. Your cards are shown at the bottom
4. Use the keyboard shortcuts to make decisions
5. Have fun!
